## Support:

Kali-linux, Kali-nethunter, Parrot & Termux
<br>
## Installation :
 
Note: Kali-linux & Parrot OS user must be use root terminal.
 
* `apt update` 
* `apt upgrade -y` 
* `apt install git -y`
* `cd $HOME`
* `git clone https://github.com/mrmani280/gamer.git` 
* `cd gamer` 
* `chmod +x setup.sh` 
* `bash setup.sh` 

## For Run : 
* `gamer`
